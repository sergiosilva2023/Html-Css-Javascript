Curso de Desevolvimento Web Full-Stack

Apresentação

O Laboratório de Inovação e Tecnologia (LabTec) da UDESC Balneário Camboriú oferecerá o curso Desenvolvedor Web full stack. O curso tem como objectivo preparar jovens para o mercado de desenvolvimento de software, fornecendo formação básica para futuros desenvolvedores (Devs), que poderão atuar em empresas de diversos segmentos do mercado.

Condições para participação

Podem se inscrever para participar do processo selectivo para o curso pessoas:

  • Maiores de 16 anos
  • Que tenham completado o Ensino Médio ou que estejam cursando o último ano de Ensino Médio
  • Que disponham de computador com acesso á internet, com banda larga fixa ou móvel de ao menos 4mb de velocidade
